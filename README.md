# m-home-resource

## 项目简介

`m-home-resource` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
这些页面通常作为静态资源或临时展示页面使用，不针对任何特定域名或网站。  
仓库内容以 HTML 文件为主，结构简单，便于直接部署或托管。

## 目录结构

```
m-home-resource/
├── pages/          # 存放各类 HTML 页面文件
├── assets/         # 公共资源文件（如 CSS、JS、图片等）
├── README.md       # 本文件
└── .gitignore      # Git 忽略规则
```

- `pages/`：每个 HTML 文件独立存放，文件名与页面用途对应。
- `assets/`：存放多个页面可能共用的样式、脚本或静态资源，避免重复。

## 页面归档说明

本仓库中的 HTML 页面主要用于归档与展示，常见用途包括：

- 临时信息页面
- 功能演示页面
- 静态内容展示
- 测试或开发用页面

每个页面均为独立可运行的 HTML 文件，不依赖外部动态服务。  
页面内容可能随时间更新或移除，建议以仓库当前版本为准。
